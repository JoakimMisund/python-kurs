10 - Nettverk (Web)

10.1 Repetisjon HTTP

Med python kan man enkelt bruke forskjellige nettverksprotokoller.
Stor mulighet for skripte foresprsler og dermed automatisere f.eks testing over nettverket.

Frst ser vi p HTTP som vi kan bruke til snakke med webservere.

  • Kan sende og motta foresprsler over HTTP med Python.
  • HTTP Metoder
    • GET - Hent data/nettside
    • POST - Last opp/send data
    • HEAD - Kun hent HTTP-header fra webserver
    • PUT - Last opp data/oppdatere enkelte ressurser
    • PATCH - Endre enkelte felter/data for en gitt ressurs
    • DELETE - Slett data

Flere moduler for HTTP:

  • Urllib
  • Urllib2
  • Urllib3
  • Requests for HTTP
  • Twisted

10.2 Surfing p web med urllib

Urllib-modulen tilbyr forskjellige funksjoner for behandle URLer og tilhrer pythons standardbibliotek. request.urlopen gjr selve pningen, og returnerer et objekt som ligner en fil, med f.eks. read(), readline() og readlines().

In[]:
from urllib import request

sock = request.urlopen('https://www.python.org')
webpage = sock.read()
sock.close()

print(webpage[:30]) # Svaret kommer som bytestreng uten kjent enkoding
print()
print(webpage.decode('utf-8')[:30]) # Som regel utf-8

Videre metoder i urllib

Noen forskjeller fra filer:

  • Ingen seek()
  • geturl() returnerer url for kallet
  • headers inneholder informasjon om server og svaret
In[]:
import urllib

with urllib.request.urlopen('https://www.python.org') as f:
    print(f.geturl() + '\n')
    print('Server: ' + f.headers['Server'])
    print('Content-Type: ' + f.headers['Content-Type'])

Utvidet bruk av urllib

Urllib kan ogs gjre mer kompliserte foresprsler, hvor man konfigurerer mer av det man nsker sende.

Man lager da frst et Request-objekt, og setter headers etter nske. Aktuelle kandidater kan vre f.eks. 'User-Agent', 'Referer' og 'Accept-Encoding'.

In[]:
from urllib.request import urlopen, Request

request = Request('https://www.python.org')
request.add_header('User-Agent', 'Mozilla/5.0 Firefox/2.0.0.11')

print(urlopen(request).read().decode('utf-8')[:500])

10.3 http.server

  • Modul fra standardbiblioteket for sette opp en enkel webserver
  • Kan brukes til gjre filer tilgjengelig over nettverket
  • Veldig enkelt sette opp
    1. G til en nsket mappe du vil gjre tilgjengelig over HTTP
    2. Kjr modulen fra kommandolinja
      • For Windows: py -3 -m http.server <PORT>
      • For Linux: python3 -m http.server <PORT>
    3. Besk http://<Din IP-addresse>:<PORT> for aksessere filene

10.3 Oppgaver: Hent noen filer fra din egen webserver

  • Bruk http.server til gjre en mappe tilgjengelig over nettverket.
  • Besk webserveren med nettleseren din og last ned noen filer manuelt.
  • Bruk python og urllib til laste ned filer fra webserveren, og skrive de til disk med riktig filnavn

JSON

Javascript Object Notation

  • En mte formidle datastrukturer i tekst.
  • Brukes blant annet for la websider hente data for oppdatere deler av en webside da det er lett parse i javascript.
  • Brukes ogs i webapi for levere data.
  • Python har en JSON modul for tolke JSON-strenger til datastrukturer og for serialisere strukturer til JSON.
In[]:
import json
my_dict= {'editor': 'Emacs', 'OS': 'Windows', 'Drink': 'Coffee', "Number": 1337}
print(my_dict, type(my_dict))
print()

json_string = json.dumps(my_dict)
print(json_string, type(json_string))
print()

new_dict = json.loads(json_string)
print(new_dict, type(new_dict))

Requests

  • Det finnes utrolig mange ulike HTTP-moduler.
  • Den som blir mest brukt (og som for mange er den nye standarden) er requests.
    • "HTTP for Humans"
    • Ikke i standardbiblioteket.
    • M installeres gjennom pip.
    • Et enklere bibliotek for gjre http-foresprsler.
    • Requests er bygget p toppen av urllib3.

  • Har mye funksjonalitet.

  • Kan enkelt sende mange typer foresprsler.

  • Returnerer et Response-objekt nr man gjr en request.

  • GET - requests.get(<URL>)

  • POST - requests.post(<URL>, data=<DATA>)

  • DELETE - requests.delete(<URL>)

  • osv..

In[]:
import requests
import json
r = requests.get('https://api.github.com/events')
print(r.headers)

print(r.headers["server"])
print(r.status_code)
In[]:
print(len(r.json()))

Bruk feltet under til se p resultatet i r, se om du kan liste ut brukernavnet for hver hendelse.

Tips: For-lkker, keys(), type(variabel), r.json()

d = r.json()

10.4 Oppgaver: Bruke API

Scriptet api.py kan brukes til starte en enkel api tjeneste med Fastapi.

Fastapi er en enkel og rask mte sette opp et api p i Python. Dokumentasjon https://fastapi.tiangolo.com/

For kjre tjenesten bruker vi her en minimal webserver kalt Uvicorn.

G til script mappen og start

In[]:
uvicorn api:app

Du vil da f et enkelt api kjrende p port 8000.

  • http://localhost:8000
  • http://localhost:8000/docs for gui

Api'et bruker text.txt fila vi brukte i filhnterings modulen av kurset.

Get funkjsonen leser fra fila og put appender til fila.

  • G til apiet sitt gui og prv funksjonene
  • Utvid Put funksjonen til kunne velge mellom append og write til fil
  • Hvilke andre metoder kan du utvide API'et med

Ekstra velse: Wikipedia-sk

Wikipedia har et API for kjre sprringer mot siden. vingen gr ut p lage et script som bruker requests for gjre sk.

  • Referanse: https://en.wikipedia.org/w/api.php

  • Eksempel: https://en.wikipedia.org/w/api.php?action=opensearch&search=skestreng

  • Etter ha listet ut sidene som treffer skestrengen bruk parse-apiet for finne de sidene som finnes p norsk. (langlinks) https://en.wikipedia.org/w/api.php?action=parse&page=Cube&prop=langlinks

  • Bruk argparse for ta inn skestrengen, f.eks med --query

  • Gjr det mulig velge sprk med lang

  • Husk at requests.get lar deg spesifisere en dict med parametre. requests.get("https://en.wikipedia.org/w/api.php",params={"action":"opensearch","search":my_search})

Tips:

  • Hvis du har flere lists/array du vil behandle samtidig: Se p zip()
  • Wikipedia formaterer api-resultat litt forskjellig. Legg til format=json for maskinlesbar og format=jsonfm for pretty-print.
